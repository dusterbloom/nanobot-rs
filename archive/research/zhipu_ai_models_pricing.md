## TECHNICAL SPECIFICATIONS

### Context Windows
- **GLM-5**: 204,800 tokens (204.8K)
- **GLM-4.7**: 202,752 tokens (202.7K)
- **GLM-4.6**: 202,752 tokens (202.7K)
- **GLM-4.5**: 131,072 tokens (131K)
- **GLM-4.5-Air**: 131,072 tokens (131K)
- **GLM-4.5V**: 65,536 tokens (65.5K)

### Architecture
- **GLM-5**: 200K context, latest flagship
- **GLM-4.7/4.6/4.5**: Mixture of Experts (MoE) architecture
  - GLM-4.5: 355B total / 32B active parameters
  - GLM-4.5-Air: 106B total / 12B active parameters

### Features
- **Prompt Caching**: Reduces costs on repeated queries (cached input pricing)
- **Thinking Mode**: Extended reasoning for complex tasks (same pricing)
- **Tool Use**: Native support for function calling
- **Streaming**: Full streaming support
- **OpenAI-Compatible**: Works with OpenAI SDK
